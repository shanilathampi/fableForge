# fableForge
An AI-powered interactive fiction engine

This single-page project uses:

Gemini (JSON Mode) to act as the Dungeon Master, managing the story, health points, and inventory state in structured JSON.

Imagen 3 to dynamically generate a visual scene for every turn of the story.

Tailwind CSS for an immersive, dark-mode "Storybook" aesthetic.

It features a robust game loop that remembers your previous choices to build a continuous narrative.
